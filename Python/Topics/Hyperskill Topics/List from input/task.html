List from input

Use the template below to write a program that takes n numbers as input, creates a list from them, and then prints it.

The first number in the sample input shows how many elements your list should contain. The numbers after it are the elements that you need to append to your list.

At this point you may not know how to add elements to a list. You can use the command list.append(element) to do that, where list is the name of your list, and element is the name of the variable that you want to add to the list.


Sample Input:
5
100
-1
72
0
0

Sample Output:
[100, -1, 72, 0, 0]



Memory limit: 256 MB
Time limit: 15 seconds
