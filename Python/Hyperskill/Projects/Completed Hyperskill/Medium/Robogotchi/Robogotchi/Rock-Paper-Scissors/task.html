Description

Two games are better than one. Let's add another: a classic game called Rock-paper-scissors.

The game is simple: the user makes their choice of weapon, and the robot's choice is generated. Their moves are then compared to determine the winner. Rock beats scissors but loses to paper, paper beats rock but loses to scissors, and scissors beat paper but not rock. As before, inputting exit game finishes the program.

Do not let the user cheat: "matches", "ponies", or "swords" are not valid options! If the user enters an invalid option, print No such option! Try again!.

Just like in the previous stage, keep the statistics and print it when the user decides to exit the game. The format remains the same:

You won: [number of your wins],
Robot won: [number of the robot's wins],
Draws: [number of the draws].

Finally, since now we have not just one game but two, let the user choose which game to play.

Your program must be case-insensitive.

Objectives

In this stage, your program should:

  1. Prompt the user to pick a game with a message Which game would you like to play?.
  2. If they choose "Numbers", proceed with the game as in the previous stage. If they choose rock-paper-scissors, implement steps 3-6. If the user input is anything else than "Numbers" or "Rock-paper-scissors", print Please choose a valid option: Numbers or Rock-paper-scissors?.
  3. Take the user input, check its validity, and ask for a valid input if needed.
  4. Output the robot's move: The robot chose [paper/rock/scissors].
  5. Output the result using an appropriate message: You won!, The robot won!, or It's a draw!.
  6. Keep taking input as long as the user wants to play.
  7. Keep the game statistics and print it when the user decides to exit the game.
Examples

The greater-than symbol followed by a space (> ) represents the user's input. Note that it's not part of the input.

Example 1

Which game would you like to play?
> Rock-paper-scissors

What is your move?
> Paper
The robot chose paper
It's a draw!

What is your move?
> rock
The robot chose scissors
You won!

What is your move?
> gun
No such option! Try again!

What is your move?
> scissors
The robot chose paper
You won!

What is your move?
> exit game

You won: 2,
The robot won: 0,
Draws: 1
