Description

In this stage, we will refer to other scientific approaches to readability.

Take one more look at different grade levels/age brackets and corresponding scores. You can use the Wikipedia article on ARI. The table is suitable for all the algorithms described in this stage. To calculate the age, use the upper bound range. For example, if the range is 12-13 year olds then the upper bound is 13 year olds.

The first algorithm is the Flesch-Kincaid Readability Test. First, you need to create a function that calculates the number of syllables in a word. The formula is given below. You can find more information about the F-K tests in the article by Readability Formulas. For calculating age, use the same table from the Automated Readability Index.

\(score = 0.39 * {words \over sentences} + 11.8 * {syllables \over words} - 15.59\)

The next one is the Coleman-Liau Index. The formula is given below. The age brackets you can find in the table from the previous stage. L is the average number of characters per 100 words, S is the average number of sentences per 100 words. Like with other indexes, the result is a minimum grade level required to understand this text. Polysyllables are words that have more than 2 syllables.

\(score = 0.0588 \ ∗ \ L−0.296 \ ∗ \ S\ −\ 15.8\)

In this stage, implement all these approaches to your program. Don't forget about the Automated Readability Index! You should also implement an option to choose all the methods at the same time.

To count the number of syllables, use the letters a, e, i, o, u, y as vowels. You can use these simple rules to count the syllables:

1. Count the number of vowels in the word.
2. Omit the so-called double vowels, that is, two vowels that follow each other. For example, "rain" has 2 vowels but only 1 syllable.
3. If a word contains 3 vowels in a row (i.e. 'notorious'), these triple vowels equal to 2 syllables.
4. Subtract the silent vowels (that is, "e" in "side" is silent, because the word ends with it, so the word has only 1 syllable).
5. If it turns out that the word contains 0 vowels, then consider this word a monosyllable.

Objectives
  1. Read the the file named "in.txt". The file is given as a command line argument.
  2. Count the number of characters, syllables, polysyllables, words, and sentences in the text.
  3. Add an option to choose the metrics to display.
  4. Count the chosen scores according to the formulas above.
  5. Count the average age.
  6. Print the information in the way, shown in the examples.
Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.
