Boolean search

Below you will see four searches. What you should do is understand whether they will return True or False and then treat the sequence as a number in the binary system.

The answer will be this number in the decimal system (for example, False True True False = 0110 = 6).

print('' in 'universe')                            # 1

pet = "myguineapignamedSparks"
print(pet.startswith("pig", 3))                    # 2

longest_word = "Antidisestablishmentarianism"
print(longest_word.endswith("establishment", 20))  # 3

person = "lover"
print(person.endswith("over"))                     # 4

For you to remember the binary system, here's the beginning of the conversion table from the decimal to the binary:
