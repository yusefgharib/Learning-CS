color_list = ['cyan', 'magenta', 'yellow', 'key color']

cmyk_file = open('CMYK.txt', 'w', encoding='utf-8')
cmyk_file.writelines(color_list)
cmyk_file.close()

What will be the contents of the file CMYK.txt?
